# АнтиМакс — Защита ваших сообщений 🔒

**АнтиМакс** — это расширение для браузера Chrome (в будущем - iOS/android), которое позволяет вам шифровать и расшифровывать сообщения для переписки в популярных мессенджерах (ВКонтакте, Макс, Telegram). Ваши сообщения защищены, а получатель может их прочитать только с помощью правильного ключа шифрования.

## 🔑 Основные возможности

- **Создание профилей** — настройте персональные профили для разных контактов
- **Шифрование сообщений** — преобразуйте текст в зашифрованный вид с помощью ключа
- **Расшифровка сообщений** — верните текст из зашифрованного вида
- **Импорт/Экспорт данных** — переносите настройки между устройствами через QR-коды или файлы
- **Автоматическое шифрование** — включите автошифрование для чатов, чтобы сообщения обрабатывались автоматически

## 🚀 Установка

### Из Chrome Web Store (рекомендуется)

Пока недоступно.

### Вручную

1. Скачайте расширение или клонируйте репозиторий, разархивируйте папку
2. Откройте `chrome://extensions/` в браузере Chrome
3. Включите **Режим разработчика** (Developer mode)
4. Нажмите **Загрузить распакованное расширение** (Load unpacked)
5. Выберите разархивированную папку

## 📖 Как использовать

### 1. Окно расширения

При первом запуске расширения создаётся ваш профиль.
Генерируются ключи шифрования.

### 2. Пригласите собеседника в секретный чат

На странице с чатом нажмите на серый замочек, так вы пригласите собеседника в секретный чат.
Или нажмите поделиться в окне расширения, скопируйте ключ и отправьте собеседнику для ручного создания чата.

### 3. Создайте чат с собеседником после приглашения

Когда вам отправлено приглашение в диалоге, то вы можете нажать кнопку "Создать чать".
Или самостоятельно создайте чат в окне расширения, введя ключ, который вам прислали.

### 4. Шифруйте сообщения

Используйте замочек в диалогах, чтобы появилось зеленое поле ввода.
Так вы защищаете свои сообщения от прослушивания.
Или самостоятельно выберите диалог в расширении и с помощью него зашифруйте текст и отправьте собеседнику.

### 5. Дешифрование сообщений

Если выбрана настройка "Автошифование", то при загрузке веб-страницы с чатом все сообщения автоматически дешифруются, и становится активным защищенное поле ввода.

### 5. Перенос между устройствами

Поддерживается экспорт и импорт.

## 🔐 Безопасность

- Расширение не отправляет ваши данные на серверы

## 🛠️ Разработка

### Структура проекта

```
extension/
├── content.js         # Основной скрипт для встраивания в страницы
├── manifest.json      # Манифест расширения
├── popup.html         # Интерфейс расширения
├── popup.js           # Логика popup-окна
├── styles.css         # Стили интерфейса
├── qrcode.min.js      # Библиотека для QR-кодов
└── scripts/
    ├── chat.js        # Управление чатами
    ├── crypto.js      # Криптографические функции
    ├── current_profile.js  # Текущий профиль
    ├── load_profiles.js    # Загрузка профилей
    ├── profile.js          # Управление профилями
    ├── profile_manager.js  # Менеджер профилей
    └── save_profiles.js    # Сохранение профилей
```

### Требования

- Chrome 109+
- Манифест версии 3 (MV3)
